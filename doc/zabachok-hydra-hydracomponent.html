$resolutions public property

Allowed resolutions.

You can use pattern or name.

By pattern:

//in config
'resolutions' => [
    '100x100p'
],
//in your code
Yii::$app->hydra->getCachePath('image.jpg', '100x100p');

By name:

//in config
'resolutions' => [
    'newsPreview' => '100x100p',
],
//in your code
Yii::$app->hydra->getCachePath('image.jpg', 'newsPreview');//It will use 100x100p

See also getCachePath().

public array $resolutions = ['preview' => '100x100p', 'newslist' => '150x150i', 'newsview' => '300x300i', '130x120p']

getCachePath() public method

Generates path to resized image.

You can set $resolution:

  1. In format 'WidthxHeightCroptype' (example: 150x100p).
  2. By name (example: 'newsView').

All resolutions must be setted in HydraComponent::resolutions.

Croptype must be 'p' or 'i'. 'p' means that image will cropped proportionally. 'i' means that image will inscribed to resolution.

See also $resolutions.
